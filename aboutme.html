About the site

I've been working on this site since 2024, although I don't update it super often because I'm lazy. The idea for it came up in my first year of learning programming. I was struggling to decide what builds to play in Deepwoken, so I decided to make a simple roulette-style tool for myself. Over time, that small idea slowly turned into a fully working website that I hope other people will actually find useful too. I won't lie, the code isn't perfect or fully optimized. A lot of it was written when I had much less experience, and this is also my first bigger project like this. I' still learning and improving it as I go. If you're interested, feel free to join the Discord so we can build a small community around it and improve the project together.
